Closure in Javascript

A closure is the combination of a function bundled together (enclosed) with references to its surrounding state (the lexical environment).

A function bind together with its lexical environment i.e Closure with its lexical scope is bundled together.

Function along with its lexical scope makes a Closure

Lexical environment => In JavaScript, a lexical environment is a data structure that stores all variables and function declarations. It allows the interpreter to recognize which variables or functions are accessible in different scopes within your program.
